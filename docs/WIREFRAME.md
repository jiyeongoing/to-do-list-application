# Swipe Todo - Wireframe

## MVP 화면

![Swipe Todo MVP Wireframe](assets/wireframe-overview.svg)

| 화면 | 사용 목적 | 핵심 행동 |
| --- | --- | --- |
| 오늘 | 당장 실행할 일을 확인 | 추가, 완료, 삭제 |
| 데일리 | 매일 반복되는 루틴을 등록 | 활성화, 비활성화 |
| 계획 | 미래의 특정 날짜에 할 일을 예약 | 월/날짜 선택, 추가, 삭제 |
| 목적별 리스트 | 장보기나 여행 준비 목록만 확인 | 항목 추가, 완료, 삭제 |

## 이동 방식

- 앱을 열면 `오늘` 화면이 먼저 보입니다.
- 기본 화면에서는 `오늘 / 데일리` 두 탭만 이동합니다.
- 기본 화면을 옆으로 넘기면 `계획` 화면이 보입니다.
- 계획 화면에서 `이번 달 / 다음 달 / 다다음 달`을 고르고 날짜는 좌우 화살표로 이동합니다.
- 선택 날짜가 되면 저장된 항목은 `오늘`로 자동 이동합니다.
- 선택 날짜의 목적별 리스트는 오늘 화면에 카드로 도착하며, 탭하면 단독 목록 화면으로 이동합니다.
- 오늘의 목적별 리스트와 데일리 루틴은 위아래로 순서를 바꿀 수 있습니다.

## 화면 설계 의도

- 자주 사용하는 실행 목록과 루틴만 기본 화면에 남겨 가볍게 만듭니다.
- 주 단위가 아니라 원하는 날짜에 바로 입력해 계획 단계를 줄입니다.
- 미래 계획은 별도 화면에 두어 오늘 할 일에 집중할 수 있게 합니다.
- 상황별 준비물은 독립된 리스트로 묶어 일반 할 일과 섞이지 않게 합니다.
