# Swipe Todo - Wireframe

## MVP 화면

![Swipe Todo MVP Wireframe](assets/wireframe-overview.svg)

| 화면 | 사용 목적 | 핵심 행동 |
| --- | --- | --- |
| 오늘 | 당장 실행할 일을 확인 | 추가, 완료, 삭제 |
| 이번 주 | 이번 주 안에 처리할 일을 정리 | 오늘로 이동 |
| 데일리 | 매일 반복되는 루틴을 등록 | 활성화, 비활성화 |
| 다음 주 | 미래 계획을 미리 메모 | 추가, 삭제 |

## 이동 방식

- 앱을 열면 `오늘` 화면이 먼저 보입니다.
- 상단 탭으로 `오늘 / 이번 주 / 데일리`를 이동합니다.
- 기본 화면에서 오른쪽으로 스와이프하면 `다음 주` 화면이 보입니다.
- 다음 주가 시작되면 저장된 항목은 `이번 주`로 자동 이동합니다.

## 화면 설계 의도

- 오늘 할 일이 가장 먼저 보여 실행에 집중합니다.
- 반복 루틴은 데일리로 분리해 매번 입력하지 않습니다.
- 다음 주 계획은 평소에는 숨겨 현재 목록이 복잡해지지 않게 합니다.
